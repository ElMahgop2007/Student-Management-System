# 🎓 Student Management System
A console-based application built with **C++** to manage student records. This project demonstrates core programming concepts like **Structs**, **Arrays**, and **Modular Programming**.

### 🚀 Features
* **Add Students:** Register new students with unique IDs.
* **Search:** Find students quickly by their ID.
* **Statistics:** Calculate class average and pass/fail rates.
* **Update:** Modify student grades on the fly.

### 🛠️ Technologies Used
* Language: C++
* Development Environment: WSL / Ubuntu
